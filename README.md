# Detection-ofHate-Crime-AmongSociety-onTwitter-usingNa-ve-Bayes-Algorithm
Twitter is a very popular social media platform. It has 28.3 million active users and most users aged 25-34 (38.5%). There are 56.4% men and 43.6% women. Hate speech on Twitter leads to hate crimes. The focus is to use the Naive Bayes algorithm to address this issue. This study aims to contribute to a safer online environment for everyone.

• Designed a Naïve Bayes-based system to detect hate speech on Twitter.
• Addressed data inconsistencies using tokenization and stop-word elimination.
• Enhanced social media safety by aiding researchers in identifying harmful content.
